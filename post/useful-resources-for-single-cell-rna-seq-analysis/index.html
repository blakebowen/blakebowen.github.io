Useful resources for single cell RNA-seq analysis

Last updated on May 31, 2022 1 min read

Introduction

I have been working on single nuclei RNA-seq and single nuclei ATAC-seq data for about the last year and a half. This is a compilation of resources I have found very helpful throughout this time.

Tutorials

Cellranger
The Cellranger pipeline is used to process raw 10x Genomics Chromium single-cell RNA-seq output into a gene expression matrix.

Seurat guided clustering tutorial
For downstream analysis, the best place to start is the vignettes published by the Satija Lab, the authors of Seurat. These tutorials provide an overview of the methods and can be used as an initial analysis framework that can be modified to analyse new datasets.

HBC Single cell RNA-seq data analysis workshop
This workshop made by the Harvard Chan Bioinformatics Core has freely available workshop materials. These workshops provide an in-depth overview of single cell analysis.
